Each word embedding contains compressed semantic and syntactic information. The transformer converts every embedding into Query, Key, and Value vectors. For a given word, the Query represents what information it is searching for, the Keys of all words represent what information they can offer for matching, and the Values contain the actual information that can be transferred. By comparing the current Query with all Keys, the transformer determines which words are relevant. The resulting attention scores are then used to combine the corresponding Value vectors, producing a context-aware representation of the current word.

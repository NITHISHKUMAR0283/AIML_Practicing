# Linear Regression

Finding a line which matches with the distribution of features given to the model

Using that line we try to predict the unseen data

---

### Model

```id="lr1"
y = w1x1 + w2x2 + ... + wnxn + b
```

---

### Loss Function (Mean Squared Error)

1. For reducing the loss, we try random values at first

2. Calculate the predicted value for all the rows in the data

3. Measure the vertical error:

```id="lr2"
(y - y_pred)   → residual
```

4. Square this error:

* to get magnitude
* to punish larger errors more

5. Add all errors for individual rows and find the mean

---

👉 This loss function forms a **bowl-shaped geometry (convex curve)**

---

### Gradient Descent (works on derivatives)

1. Find the change in weights and bias:

```id="lr3"
dw = ∂L/∂w  
db = ∂L/∂b
```

2. Update parameters:

```id="lr4"
w = w - lr * dw  
b = b - lr * db
```

3. Gradient gives the direction of **steepest increase of loss**

* we take the opposite direction
* and move downward
* reaching the minimum possible value

---

### Final Outcome

We obtain the most suitable:

```id="lr5"
w1, w2, ... , wn and b
```

---

### Prediction

Using the learned weights and bias:

```id="lr6"
y_pred = w1x1 + w2x2 + ... + wnxn + b
```

👉 We can now predict values for unseen data

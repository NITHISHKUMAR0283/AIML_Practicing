# Linear Regression

Finding a line which matches with the distribution of features given to the model

Using that line we try to predict the unseen data

---

### Model

```id="lr1"
y = w1x1 + w2x2 + ... + wnxn + b
```

---

### Loss Function

Loss function applied on:

```id="lr2"
y = w1x1 + w2x2 + ... + wnxn + b
```

1. For reducing the loss we try random values at first

2. Calculate the value for all the rows in the data

3. Measure the vertical error:

```id="lr3"
(y - y_pred)   → residual
```

4. Square this:

* to get magnitude
* to punish greater error to greater extent

5. Add all errors for individual rows and find the mean

---

👉 This loss function is a **bowl-shaped geometry (convex)**

---

### Gradient Descent (works on derivative)

1. Find the net change of weights and bias:

```id="lr4"
dw = ∂L/∂w  
db = ∂L/∂b
```

2. Update the parameters:

```id="lr5"
w = w - lr * dw  
b = b - lr * db
```

3. Gradient gives the direction of **steepest increase of loss**

* we take the opposite direction
* make it move downwards
* reaching the minimum value possible

---

### Final Outcome

Finally, we get the most suitable:

```id="lr6"
w1, w2, ... , wn and b
```

---

### Prediction

With the above vector ( w ) and ( b ), we can predict the values of unseen data:

```id="lr7"
y_pred = w1x1 + w2x2 + ... + wnxn + b
```
